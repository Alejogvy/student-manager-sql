# Student Manager SQL

## 📌 Description

This project is a Student Management System developed using JavaScript and SQLite.

The application runs in the terminal and allows a teacher to manage students, subjects, and grades. It demonstrates how to use SQL to interact with a relational database.

---

## 🚀 Features

- Add students
- View all students
- Update student information
- Delete students
- Add subjects to students
- Add grades to subjects
- Calculate average grades per subject
- Calculate average grades per student
- Delete subjects and grades
- View all data using SQL JOIN

---

## 🧠 Technologies Used

- JavaScript (Node.js)
- SQLite (Relational Database)
- SQL (queries: INSERT, SELECT, UPDATE, DELETE, JOIN, AVG)
